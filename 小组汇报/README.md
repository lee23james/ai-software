# 小组汇报材料索引

本目录用于准备课堂汇报中关于原型、MVC 组件、组件图、部署图、包图、安全与性能等问题的回答材料。

当前项目已经完成基础框架搭建，包括：

- Vue 3 前端可操作原型。
- Spring Boot 后端三层架构骨架。
- FastAPI AI 服务骨架。
- 前端、后端、AI 服务三端可本地启动。

需要注意：当前材料中的图以 Mermaid 和文本线框图形式编写，适合直接复制到支持 Mermaid 的 Markdown 工具中展示，也可以根据这些内容在 ProcessOn、draw.io、Figma 中重画为图片。

## Mermaid 导出建议

可以使用以下方式将 Mermaid 图导出为图片：

1. 使用 VS Code 的 Markdown Preview Mermaid Support 插件预览。
2. 使用 Typora 直接预览 Mermaid。
3. 复制 Mermaid 代码到 Mermaid Live Editor。
4. 按照 Mermaid 内容在 ProcessOn / draw.io 中重画。
