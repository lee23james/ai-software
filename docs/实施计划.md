# 基础框架实施计划

## 1. 目标

搭建一个可以直接启动和继续开发的课程项目基础框架，包括 Vue 前端、Spring Boot 后端、FastAPI AI 服务、基础文档、示例数据和启动脚本。

## 2. 实施步骤

1. 创建前端工程骨架。
2. 创建后端工程骨架。
3. 创建 AI 服务工程骨架。
4. 添加项目级文档、示例数据和启动脚本。
5. 运行基础测试或构建命令，确认框架可用。

## 3. 验收标准

- `frontend` 可以安装依赖并通过路由测试。
- `backend` 可以通过 Maven 测试，健康接口可用。
- `ai-service` 可以通过 Pytest 测试，健康和匹配接口可用。
- README 中包含清晰的启动方式。
- 三端目录边界清晰，方便团队成员分别继续开发。
