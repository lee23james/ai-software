# 接口说明

## 1. 统一返回格式

后端接口统一返回：

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

AI 服务接口返回结构保持简洁，后续由 Java 后端统一封装给前端。

## 2. 后端接口

### 2.1 健康检查

```http
GET /api/health
```

示例响应：

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "service": "backend",
    "status": "UP"
  }
}
```

### 2.2 岗位列表

```http
GET /api/job/list
```

当前返回模拟数据，后续接入数据库分页查询。

### 2.3 城市岗位统计

```http
GET /api/analysis/city-job-count
```

当前返回 ECharts 可直接消费的基础数组。

### 2.4 管理员看板

```http
GET /api/admin/dashboard
```

当前返回岗位数、用户数、简历数等模拟统计值。

### 2.5 简历详情（含岗位选择建议字段）

```http
GET /api/resume/{resumeId}
```

响应 `data` 中除原有字段外，包含：

- `jobSelectionAdvice`：由 DeepSeek 生成的自然语言岗位选择建议（未生成时为 `null`）
- `jobSelectionAdviceModel`：生成建议时使用的模型名
- `interestResumeAdvice`：针对用户在「兴趣岗位」接口中保存的意向岗位生成的简历修改建议（未生成时为 `null`）
- `interestResumeAdviceModel`：上述建议使用的模型名

### 2.6 生成岗位选择建议（DeepSeek，Java 后端直连）

```http
POST /api/resume/{resumeId}/job-selection-advice
```

无请求体。后端会汇总该简历的解析文本、已保存技能与 **全部在招岗位**（`job_info.status=1`），调用 DeepSeek Chat Completions（OpenAI 兼容接口），将建议写入数据库并返回。

成功示例：

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "advice": "……自然语言建议……",
    "model": "deepseek-chat"
  }
}
```

**环境变量**：后端需配置 `DEEPSEEK_API_KEY`。未配置时 `code` 为 `503`。

**其它错误**：调用 DeepSeek 失败时 `code` 为 `502`，`message` 为可读说明。

更多参数见 `backend/src/main/resources/application.yml` 中 `deepseek.*`（如 `model`、`read-timeout`）。

### 2.7 按兴趣岗位生成简历修改建议（DeepSeek）

兴趣岗位名称通过既有接口维护：

```http
POST /api/user/interest-jobs
GET /api/user/interest-jobs?userId=
```

（数据存于用户档案 `target_position`，逗号分隔。）

生成建议：

```http
POST /api/resume/{resumeId}/interest-resume-advice
```

无请求体。后端根据简历所属用户读取已保存的兴趣岗位列表，并尽量附带 `job_info` 中与意向名称相关的在招岗位描述（截断后）作为上下文，调用 DeepSeek 输出**针对这些意向岗位的详细简历修改建议**，写入 `resume_info.interest_resume_advice` 并返回，响应体形状同 2.6（`data.advice` / `data.model`）。

若该用户尚未保存任何兴趣岗位，`code` 为 `400`，提示先保存兴趣岗位。

## 3. AI 服务接口

### 3.1 健康检查

```http
GET /health
```

### 3.2 简历解析

```http
POST /ai/resume/parse
```

请求体：

```json
{
  "resumeText": "熟悉 Java、Spring Boot、Vue 和 MySQL。"
}
```

### 3.3 岗位匹配

```http
POST /ai/job/match
```

请求体：

```json
{
  "resumeText": "熟悉 Java、Spring Boot、Vue 和 MySQL。",
  "jobDescription": "要求掌握 Java、Spring Boot、MySQL，了解 Redis。"
}
```

### 3.4 简历优化建议

```http
POST /ai/resume/suggestion
```

请求体：

```json
{
  "resumeText": "做过课程设计。",
  "targetJob": "Java 后端开发实习生"
}
```
